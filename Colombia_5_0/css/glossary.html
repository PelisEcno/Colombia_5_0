Glosario técnico

Término Definición
PhishingTecnica de engaño para robar datos personales mediante correos o paginas falsas
RansomwareMalware que bloquea archivos y pide dinero para liberarlos
CortafuegosSistema que filtra el trafico de red para bloquear accesos no autorizados
CifradoProceso de convertir informacion en un codigo para protegerla
VulnerabilidadFallo en un sistema que puede ser aprovechado por atacantes
Software maliciosoPrograma diseñado para dañar o infiltrarse en un sistema sin permiso
Interfaz de usuarioConjunto de elementos visuales con los que el usuario interactua
Experiencia de usuarioComo se siente el usuario al interactuar con un producto digital
Visualizacion en pantallaElementos de interfaz que muestran informacion sin interrumpir la accion
Esquema de diseñoBoceto basico de como se va a ver una pantalla o interfaz
Computacion en la nubeAcceso a servicios y recursos de computo a traves de internet
Inteligencia artificialCapacidad de las maquinas para simular procesos del pensamiento humano
Aprendizaje automaticoRama de la IA donde los sistemas aprenden de datos sin ser programados explicitamente
Grandes datosConjuntos de datos tan grandes que no se pueden procesar de forma tradicional
AlgoritmoConjunto de pasos ordenados para resolver un problema
Interfaz de programacionConjunto de reglas que permite a dos aplicaciones comunicarse entre si
ServidorComputador que almacena y entrega recursos o servicios a otros dispositivos
Base de datosSistema organizado para almacenar y consultar informacion
Marco de trabajoConjunto de herramientas y librerias para facilitar el desarrollo de software
Codigo abiertoSoftware cuyo codigo puede ser visto, modificado y distribuido libremente
AutomatizacionUso de tecnologia para realizar tareas sin intervencion humana
CiberseguridadPracticas y tecnologias para proteger sistemas y datos de ataques digitales
TokenClave digital usada para verificar identidad o autorizar acciones
DespliegueProceso de publicar una aplicacion para que sea accesible por los usuarios
RepositorioEspacio donde se guarda y gestiona el codigo de un proyecto
Diseño adaptableTecnica para que un sitio web se vea bien en cualquier tamaño de pantalla
PrototipoVersion inicial de un producto usada para probar ideas antes del desarrollo final
Parte traseraParte del desarrollo que el usuario no ve, donde se procesa la logica del sistema
Parte delanteraParte visual de una aplicacion con la que el usuario interactua directamente
DevOpsCultura que une desarrollo y operaciones para entregar software mas rapido
